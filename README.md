NeuroTrack is a digital wellness application that monitors user health through behavioral biometrics.

Key Features
Typing Analysis: Detects mental focus by analyzing keystroke dynamics.
Screen Time Tracking: Monitors usage duration using Android System APIs.
Real-time Alerts: Provides color-coded feedback for mental and physical fatigue.

Technology Stack
Language: Kotlin
UI Framework: Jetpack Compose
Database: Room DB (Local)
API:Android UsageStatsManager
